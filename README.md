# Expense-Tracker
Expense Tracker je veb aplikacija za praćenje troškova i prihoda u poslednjih 7 dana.   Korisnik može da kreira nalog i da se uloguje. Nakon logovanja, na glavnoj stranici vidi pregled svog novca. Omogućeno je dodavanje novih transakcija po kategorijama, a kategorije korisnik može i sam da kreira.
